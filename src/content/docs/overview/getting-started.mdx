Elixir UI is a modern, minimal React component library that helps you build beautiful interfaces fast.

---

## 1. Install Dependencies


Make sure to <a target='_blank' href='https://ui.shadcn.com/docs/installation'>add shadcn in your project</a>

OR

Make sure Tailwind CSS, Framer Motion and clsx are installed:

<CodeBlock language="html" code={"npm install tailwindcss framer-motion clsx"} />
---

## 2. Add a Component

Use the CLI to add the Button component:

<CodeBlock language="html" code={"npx shadcn@latest add https://ui.elixirlabs.in/registry/button.json"} />

Or copy-paste the Button code from the registry directly.

---

## 3. Use the Component

Import and use the Button:

<CodeBlock language="html" code={`import { Button } from '@/components/ui/button'

export default function Demo() {
      return <Button className="px-4 py-2 bg-blue-600 text-white rounded">Click Me</Button>
}`} />

---

Elixir UI components are 100% customizable, so customize them easily with Tailwind CSS.
