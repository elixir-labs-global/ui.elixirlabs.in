To get started with Elixir UI, follow these simple steps to set up the required dependencies and add components to your project.

---

## 1. Install Dependencies


Make sure to <a target='_blank' href='https://ui.shadcn.com/docs/installation'>add shadcn in your project</a>

OR

Make sure Tailwind CSS, Framer Motion and clsx are installed:

<CodeBlock language="html" code={"npm install tailwindcss framer-motion clsx"} />
---

## 2. Add a Component

Use the CLI to add the Button component:

<CodeBlock language="html" code={"npx shadcn@latest add https://ui.elixirlabs.in/registry/button.json"} />

Or copy-paste the Button code from the registry.

---
